	1. introduction:

		KIU MOD is a mod made for kerbal space program, this mod aim to bring all the Chinese space rockets and space vehicle to the game.
		Through the long-term and continuous efforts of the creators, we have completed the Tianwen Project, the Chinese Space Station and the Long March series of carrier rockets and many other contents.
		The KIU MOD includes KCHS (KIU Chinese Human Spaceflight Pack), KCDE (KIU Chinese Deepspace Exploration Pack), KCLV (KIU Chinese Launch Vehicle Pack) Three module packages.

	2. User Agreement:

		There are a few things you need to know and agree on before you start using this plug-in. Continuing to install this plug-in means that you have approved the following:

			1. This mod follows the CC BY-NC-SA 4.0 policy. Plugin in partial texture, partial waterfall template, FX resources from Kerwis Chinese Aerospace Package (author :Velctor, Bingo, Akino)
				The other waterfall files are from Stock Waterfall Effects (author: AtomicTech) and SWE (SRB Waterfall Effects (author: Adiri)

			4. The content of this plug-in is entirely based on public information and some imagination, and cannot accurately restore the reality.

	3. Tips：

		1) It is recommended to turn on "Advanced Settings/Auto Hardening" to stabilize the vehicle body

				Or install the "KerbalJointReinforcement" vehicle reinforcement mod.
                                            
		2) Installation methods：
			
			A. Install manually:

				①Delete old version of "NTCP", "NCAP" and "ICSP" folder (please ignore this step if you did not install older version.)

				②Drag the "KIU" folder from the new zip package to the "GameData" folder in the game directory to complete the installation.

				③Be careful not to nest folders.
			
			B. Install with Ckan:

				①Search for "KIU", find "KCHS (KIU Chinese Human Spaceflight Pack)" and install it.
                                                                	or: "KCDE (KIU Chinese Deepspace Exploration Pack)"
                                                                	or: "KCLV (KIU Chinese Launch Vehicle Pack)"

		3) installation notes: Some players wanted to play in the original environment at full size, so we did this:

				A. If you want to play the original size vehicles on non-ro environment, please delete the "Extra" folder from the "KIU\Common" path

				B. If you want to play the smaller size vehicles on non-ro environment, you do NOT delete the "Extra" folder.

				C. If you are RO player, you need to do nothing.

中文：
	
	一、简介：

		KIU MOD是一个为坎巴拉太空计划（kerbal space program）创作的游戏插件，其目的在于为游戏添加中国航天火箭与载荷。
		经过创作者团队长久且持续的努力，我们已经完成了天问计划，中国空间站和长征系列运载火箭等诸多内容。
		KIU MOD包含KCHS (KIU Chinese Human Spaceflight Pack)，KCDE (KIU Chinese Deepspace Exploration Pack)，KCLV (KIU Chinese Launch Vehicle Pack)三个模组包。

	二、用户协议：

		在开始使用本插件之前，您需要了解并认可一些内容。继续安装本插件意味着您已认可以下内容：

			1. 本插件遵循CC BY-NC-SA 4.0协议。插件中的部分纹理，部分的waterfall模板，FX资源来自Kerwis中国航空航天包 (作者:Velctor, Bingo, Akino)
				另一部分waterfall文件来自Stock Waterfall Effects (作者: AtomicTech)以及SWE (SRB Waterfall Effects, 作者: Adiri）
				核心舱内饰来自ICSP（Icecovery's Chinese Spacecraft Pack）（作者：IcecoveryStudio、Armor）
				这些mod遵循CC BY-NC-SA 4.0或CC BY-NC-SA协议。

			2.本插件为免费插件，任何组织或个人不得将本插件的任何内容用于商业行为。

			4. 本插件的内容完全基于公开信息以及一些想象力，并不能完全准确的还原现实。

	三、提示：

		1. 建议使用“高级设置”/“自动加固”以稳定载具主体

				或安装“KerbalJointReinforcement”载具加固插件。
                                            
		2. 安装方法：
			
			A. 手动安装:

				①删除旧"NTCP"、"NCAP" 和 "ICSP" 文件夹（没有安装旧版请忽略此步骤）。

				②从新的压缩包拖动"KIU"文件夹至游戏目录的"GameData"文件夹，即可完成安装。

				③注意不要自作主张嵌套文件夹.
			
			B. Ckan安装：

				①搜索"KIU"，找到"KCHS (KIU Chinese Human Spaceflight Pack)"并安装即可。
                                                                或："KCDE (KIU Chinese Deepspace Exploration Pack)"
                                                                或："KCLV (KIU Chinese Launch Vehicle Pack)"

		3.安装注意事项：因为有些玩家想在原版环境下玩真实大小，我们采取了这样的办法：

				A.如果想要在原版玩原版大小，那么请把KIU\Common路径里的“Extra”文件删除

				B.如果想在原版玩真实大小，那么请不要删除此文件。

				C.RO玩家不需要任何操作。
